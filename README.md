# Future_DS__01
This project analyzes retail sales data using Excel to identify revenue trends, top-performing products, key customer segments, and regional performance. An interactive dashboard with KPIs and slicers was developed to support data-driven business insights and decision-making.

Key Insights:
Sales exhibit strong seasonality, with peak revenue occurring in Q4.
A small number of products contribute the majority of total revenue, following the Pareto principle.
Revenue is concentrated among a limited group of high-value customers.
The domestic market generates the highest sales, while international regions show growth potential.
Average Order Value remains stable, indicating consistent customer purchasing behavior.
